# 💸 Medical Insurance Cost Prediction

A Machine Learning approach to predicting individual medical costs billed by health insurance using regression analysis.

## 📌 Project Overview
The goal of this project is to accurately estimate the annual insurance premium for a customer based on their profile. This helps insurance companies set fair prices and helps customers understand the financial impact of lifestyle choices.

## 📊 Key Findings
After performing Exploratory Data Analysis (EDA), the following factors had the highest impact on insurance costs:
* **Smoker Status:** Smokers, on average, pay more per year.
* **BMI:** High BMI significantly increases costs, especially when combined with smoking.
* **Age:** There is a steady linear increase in costs as patients get older.
